# Scoring Notes

このフォルダは、現在の長期記憶MVPが「どの基準で入力文を評価して保存しているか」を自然言語で共有するための場所です。

## 目的

- 実装コードを直接読まなくても、現在の保存基準を把握できるようにする
- スコアや判定ルールを変更したときに、説明も同じ場所へ追記できるようにする
- 他の人が「なぜこの入力が保存されたのか」を追いやすくする

## 今の実装との対応

現在のスコア実装は [app/analysis/text_analyzer.py](/Users/kabuto/Documents/Codex/memory_project/app/analysis/text_analyzer.py) にあります。

このフォルダでは、少なくとも次を維持します。

- `current_scoring.md`
  今この瞬間のスコア定義と保存判断の考え方
- 将来必要なら `changes/` や `examples/`
  スコア変更履歴や具体例の蓄積

## 更新ルール

- スコア項目を追加・削除したら `current_scoring.md` を更新する
- 判定基準を変えたら、何を変えたかを自然言語で書く
- 実装ファイルの変数名と文書の見出し名は極力合わせる

今はまだ簡易MVPなので、説明も「現在そうしている理由」を優先して書いています。厳密な評価実験や重み調整ログはまだありません。
